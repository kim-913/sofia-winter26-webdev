Assignment: Creating a Simple HTML Form
Objective
The purpose of this assignment is to help students practice creating basic HTML forms and using different form controls to collect user input.

Instructions
Create a simple HTML web page that contains a form to collect information from a user. Your form should include the following elements:

Text input field for the user’s name

Email input field for the user’s email address

Password input field

Radio buttons to select one option (example: favorite device – Laptop, Tablet, Phone)

Checkboxes to select multiple interests (example: Music, Sports, Technology, Art)

A drop-down list (select menu) to choose a favorite color

A text area where the user can write a short comment

A Submit button to send the form

A Reset button to clear the form

Requirements
Use the <form> element to contain all form controls.

Use <label> elements to clearly describe each input field.

Organize your form neatly so it is easy to read.

Deliverables
Submit one HTML file named:

yourname_form.html

Example Idea
Your form can be a “Student Information Form”, “Survey Form”, or “Contact Form.”

Grading
Form includes all required fields – 25%

Proper use of HTML form elements – 15%

Page organization and readability – 10%
